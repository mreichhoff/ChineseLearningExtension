Chinese Learning Extension Options

Anki

Integration with Anki, via AnkiConnect, allows you to directly add anki cards from words and sentences you encounter while browsing. To enable the integration, simply install the AnkiConnect add-on in anki on your computer. Then, the info in the "Learn More" panel will include buttons to make flash cards in one click. Note that all cards are added to the deck of your choice; you'll need to have at least one deck before using the Anki integration. See Anki's guide for general information about Anki.

OpenAI

Integration with OpenAI allows you to get AI analysis of words and sentences you encounter while browsing the web.
To set up the integration, enter your OpenAI API key below.
To obtain an OpenAI API key, visit the OpenAI API website, and follow their setup instructions.
